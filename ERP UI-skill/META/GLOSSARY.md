# ERP UI 统一术语字典
统一全局叫法，避免文案、设计、开发理解偏差

| 同义别名 | 标准术语 | 释义 |
|---------|----------|------|
| 质检单/验收单 | 验货单 | 仓库货品检验单据页面 |
| 货品编码/产品ID | SKU | 商品最小库存单位 |
| 清单/数据表 | 列表页 | 多行数据展示页面 |
| 明细/资料页 | 详情页 | 单条数据完整信息页 |
| 录入页/编辑页 | 表单页 | 新增、编辑数据录入页面 |
| 下拉选择/单选框 | 选择器 | 通用选择类控件 |
| 弹窗/浮层 | 模态弹窗 | 居中弹出操作窗口 |
| 顶部菜单/侧边栏 | 导航栏 | 系统全局导航结构 |
