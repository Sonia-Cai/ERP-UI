# 查看类表单 — 画板参考与 Demo 还原说明

## 文档定位（请先读）

本文件从 Figma 中的**具体画板**拆解像素、分区与示例文案，用于：

- 做**查看类 / 只读详情**相关 **HTML Demo** 或高保真还原时对照；
- 理解「查看详情」「SKU 包装尺寸」两稿的版式差异。

**通用规则（Token、编辑/查看边界、抽屉宽度算法等）一律以 `form-standard.md` 为准**（尤其 **§2.1.3～2.1.5**、**§4.1**）。本文件**不**替代表单规范，也**不**要求所有查看页复制下列业务字段或列结构。

**两种查看类抽屉落地**（纯 KV vs 摘要 + 二级卡片）的对照表与选型说明见 **`form-standard.md` 2.1.5**；本文件各画板像素供还原时对照。

## 3. 两画板对照（便于 Demo 选型）

| 维度 | 画板一（查看详情） | 画板二（SKU 包装尺寸） |
| :--- | :--- | :--- |
| 主区侧重 | 大宽表 + 多指标 / 对照 | 筛选 + 工具栏 + **紧凑**主表（可展开） |
| 数据行高（该稿） | **98px**（富内容） | **68px**（紧凑） |
| 抽屉示例宽度（该稿） | 约 **800px** | 约 **792px** |

两档数据行高为**并存**策略，按业务信息密度选用；**不得**写死为全系统唯一行高。抽屉宽度一律受 **`form-standard.md` §4.1** 约束。
