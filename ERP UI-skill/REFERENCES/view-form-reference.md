# 查看类表单 — 画板与 Demo 数值参照

## 1 关于本文件

| 问 | 答 |
| :--- | :--- |
| **本文件是什么？** | **Figma 画板**与**静态 Demo** 的**数值、分区、链接**清单，用于高保真还原时对照布局与尺寸。 |
| **设计总则在哪里？** | **[`view-form-standard.md`](./view-form-standard.md)** — 与 [`form-standard.md`](./form-standard.md)（表单设计通用规范）并列的**查看类表单设计通用规范**。 |
| **画板与规范冲突时？** | 以 **`view-form-standard.md`** 为准；本文件表格中的 px、列名为**该稿示例**，勿写死为全系唯一常量。 |

**Demo 定位：** **`demos/`** 内 HTML 仅演示**布局与 Token 用法**，业务文案可替换；索引见 **`case-studies/good-case.md`** **§4**、**§5**。

## 2 画板一：查看详情表单

**用途示例**：整页只读宽表 + 右侧只读抽屉（绩效/多指标类信息密度）。

| 项 | 画板中可参考的取值 / 结构 |
| :--- | :--- |
| Figma | [查看详情表单](https://www.figma.com/design/EUNVfCK8meDTEw3LY8Mhmn/Untitled?node-id=17-26757)（`node-id=17-26757`） |
| 主内容区内边距 | **24px** |
| 表头行高 | **46px** |
| 数据行行高（富内容档） | **98px**（多行指标、标签、单元格内对照等） |
| 只读抽屉 | 标题栏 **56px**；该稿构图下内容区约 **800px**（**非**全局固定值，宽度须满足 `form-standard.md` **§4.1**：≤ 视口 **80%**、`min(内容, 上限)`） |
| 抽屉内多列摘要 | 约三列、单列宽约 **240px** 栅格；上标题下值，标题底到内容顶约 **12px**（与 `view-form-standard.md` **§2** 一致） |
| 抽屉内分段 | `form-standard.md` **§2.4** 一级标题条 + 一级分割线；底栏 **56px**、按钮 **32px**；底栏文案见 `view-form-standard.md` **§1.4** / **§3.2** |
| 单元格对照 | **`VS`**（或统一中文「对比」）；涨跌色用语义 Token（见 `view-form-standard.md` **§3.3**） |

## 3 画板二：SKU 包装尺寸 - 详情表单

**用途示例**：主区「列表筛选 + 工具栏 + 可展开主表」+ 右侧只读抽屉（外箱 / 包装 / 产品等分段 KV）。

| 项 | 画板中可参考的取值 / 结构 |
| :--- | :--- |
| Figma | [SKU包装尺寸-详情表单](https://www.figma.com/design/EUNVfCK8meDTEw3LY8Mhmn/Untitled?node-id=19-34505)（`node-id=19-34505`） |
| Figma · **上下布局**稿（只读摘要 + 二级卡片 KV） | [SKU包装尺寸-详情表单/上下布局](https://www.figma.com/design/EUNVfCK8meDTEw3LY8Mhmn/Untitled?node-id=20-35869)（`node-id=20-35869`） |
| 静态 HTML Demo（对齐上下稿） | [`demos/sku-packaging-dimensions-detail-view.html`](../demos/sku-packaging-dimensions-detail-view.html) |
| 业务 Tab 行 | 高约 **46px**，底部分割线拉通主内容 |
| 筛选行 | **5×** `--input-width-medium`（**240×32**），水平间距 **16px**；与表体垂直间距可与列表页统一（**16px** 或 **24px**） |
| 表格工具栏 | 左：**32px** 高按钮组；右：**18px** 图标（刷新、密度、设置），与内容区右内边距 **24px** 对齐 |
| 表头行高 | **46px**（与画板一一致） |
| 数据行行高（紧凑档） | **68px**（编码、短枚举为主；某列两行描述时文本块高约 **44px**，仍落在 **68px** 行框内） |
| 序号列 / 可展开表 | 序号列宽约 **72px**，预留 **16×16** 展开位；展开子表行高与列宽见后续 `table-standard.md` 或本文件迭代 |
| 列宽档位（该稿） | **172px / 132px** 等与 8px 栅格协调；单元格左右内边距约 **16px** |
| 单元格空值 | 稿中为 ASCII **`-`**；展示层建议与规范统一为 **`—`**（见 `view-form-standard.md` **§1.4**） |
| 只读抽屉 | 标题栏 **56px**「详情」+ 左上角关闭图标；该稿构图下约 **792px**（**非**固定值，遵守 **`form-standard.md` §4.1**）；底栏见 `view-form-standard.md` **§3.2** |
| 抽屉内分段 | **2×16** 条 + 标题；可跟弱链「填写示例」；浅灰卡片内 **24px** 内边距；KV 版式见 `view-form-standard.md` **§2**、**§4**；卡片内纵向块间距可与稿中 **16px** 对齐；字段旁可有 **14×14** 帮助图标 |

## 4 两画板对照（便于 Demo 选型）

| 维度 | 画板一（查看详情） | 画板二（SKU 包装尺寸） |
| :--- | :--- | :--- |
| 主区侧重 | 大宽表 + 多指标 / 对照 | 筛选 + 工具栏 + **紧凑**主表（可展开） |
| 数据行高（该稿） | **98px**（富内容） | **68px**（紧凑） |
| 抽屉示例宽度（该稿） | 约 **800px** | 约 **792px** |

两档数据行高为**并存**策略，按业务信息密度选用；**不得**写死为全系统唯一行高。抽屉宽度一律受 **`form-standard.md` §4.1** 约束。
