# B端 ERP 图标通用设计规范
## 一、规范总则
### 适用范围
本规范适用于万物ERP全系统：侧边导航、顶部菜单、表格操作、表单控件、空状态、弹窗、看板、按钮附属图标、状态标识等所有图标场景。

### 设计原则
- 极简商务、表意优先、风格统一
- 拒绝卡通、拒绝渐变、拒绝立体花哨风格
- 尺寸/描边/圆角/颜色全局统一，无个性化自定义
- 贴合ERP业务语义，图标和业务场景强匹配

## 二、基础视觉风格
1. 统一类型：**线性线框图标**，不使用面性、双色、渐变图标
2. 描边粗细：固定 **2px** 统一描边
3. 圆角规则：图标拐角统一小圆角，禁止直角、大圆角混用
4. 构图网格：遵循 24x24 基础网格构图，视觉重心居中
5. 透视：纯平面二维，禁止阴影、透视、立体效果

## 三、尺寸规范（只允许用这5个）
### 标准可用尺寸
- 12px：辅助小字旁极简图标、标签附属图标
- 14px：表格操作列、列表行内操作图标
- 16px：导航菜单、表单输入框附属图标、常规功能入口
- 20px：卡片标题、模块标题图标
- 24px：空状态、错误页、结果页大图标

### 禁止
禁止使用 18px、22px、26px 等非标奇数值尺寸。

## 四、色彩规范
所有图标颜色**必须沿用 design-tokens.css 变量**，不写死色值。
1. 常规默认：跟随 `--color-text-secondary` 次级文本色
2. 高亮选中：跟随 `--color-primary` 品牌主色
3. 成功状态：跟随 `--color-success`
4. 警告状态：跟随 `--color-warning`
5. 危险/删除/驳回：跟随 `--color-error`

## 五、通用使用规则
1. 表意优先，只在**有语义帮助**时加图标，不做无意义装饰
2. 同一场景图标尺寸、风格、位置严格统一
3. 文字+图标组合时：图标居左、文字居右，间距统一 `--spacing-super-tight`
4. 避免同页面出现风格不一致的混搭图标
5. 功能相近场景，复用同一图标，不重复造新图标

## 六、分场景应用规范
### 1. 侧边导航 / 顶部菜单
- 固定尺寸：16px
- 未选中：中性辅助色
- 选中/当前页：品牌主色
- 每个业务模块固定唯一专属图标，全局不混用

### 2. 数据表格
- 操作列图标统一 16px
- 编辑、查看、删除、更多 固定配套标准图标
- 禁止私自替换同义非标图标

### 3. 表单页面
- 输入框前置/后置图标：16px
- 必填、提示、帮助信息配套小图标统一规范
- 附件、上传、下载使用固定业务图标

### 4. 空状态 / 错误状态 / 无数据页
- 主图标固定 24px
- 居中展示，搭配文案，不随意换风格

### 5. 数据看板 / 数据卡片
- 卡片标题图标：20px
- 趋势、上升、下降、同比环比使用固定标准图标

### 6. 弹窗 / 确认提示
- 提示类图标统一尺寸、统一配色逻辑
- 危险操作弹窗固定使用风险类标准图标

## 七、ERP 业务固定图标映射表（强制统一）
以下业务场景**固定绑定指定语义图标**，AI/设计不得随意更换：
- 首页 / 工作台：首页图标
- 采购管理：采购/订单图标
- 库存 / 仓储：仓库/库存图标
- 验货 / 品质：质检/核验图标
- 交付 / 生产：工厂/交付图标
- 物流 / 发货：物流/卡车图标
- 运营 / 销售：店铺/销量图标
- 财务 / 对账：账单/记账图标
- 供应商管理：企业/合作图标
- 用户 / 权限：人员/角色图标
- 设置 / 配置：齿轮/设置图标
- 导出 / 下载：下载图标
- 新增 / 添加：加号图标
- 编辑 / 修改：编辑图标
- 删除 / 作废：删除图标
- 查看 / 详情：预览/查看图标

## 八、硬性禁止规则 NEVER
1. 禁止混用线性和面性图标
2. 禁止自定义描边粗细、私自改圆角
3. 禁止使用卡通、萌系、节日花哨图标
4. 禁止乱用非标尺寸
5. 禁止无意义堆砌装饰性图标
6. 禁止业务场景随意替换标准语义图标
